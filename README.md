# Techy 🚀

> Techy is a clean, lightweight, and modern blog posting website designed for technology enthusiasts to read, write, and share insightful articles seamlessly.

---

## 🛠️ Tech Stack & Language Breakdown

The project leverages a lightweight Python backend paired with a heavily customized, responsive user interface.

*   **Backend:** Python 🐍 (Flask framework)
*   **Form Handling:** WTForms / Flask-WTF (`forms.py`)
*   **Frontend UI:** CSS3, HTML5, JavaScript

According to the repository metrics, the codebase consists of:
*   🎨 **CSS:** 88.7%
*   🌐 **HTML:** 6.6%
*   🐍 **Python:** 4.3%
*   ⚡ **JavaScript:** 0.4%

##🚀 Getting Started
Follow these steps to set up and run Techy locally on your machine.

1. Prerequisites
Ensure you have Python 3.x installed on your system.
2. Clone the Repository
Bash
git clone [https://github.com/Vedrob/Techy.git](https://github.com/Vedrob/Techy.git)
cd Techy
3. Set Up a Virtual Environment (Recommended)
Bash
## Windows
python -m venv venv
venv\Scripts\activate


## macOS/Linux
python3 -m venv venv
source venv/bin/activate
4. Install Dependencies
Bash
pip install -r requirements.txt
5. Run the Application
Bash
python main.py
Once the local server initializes, open your browser and navigate to http://127.0.0.1:5000/.
